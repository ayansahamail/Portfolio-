# Ayan Saha Site Clone

This project is a clone of the website located at "https://ayan-saha-dev-8b824qs.gamma.site/". It is built using HTML, CSS, and JavaScript, and aims to replicate the design and functionality of the original site.

## Project Structure

```
ayan-saha-site-clone
├── index.html          # Main HTML file
├── README.md           # Project documentation
├── css
│   ├── styles.css      # Main styles for the website
│   └── components
│       ├── header.css  # Styles for the header component
│       ├── hero.css    # Styles for the hero section
│       ├── sections.css # Styles for various sections
│       └── footer.css   # Styles for the footer component
├── js
│   ├── main.js         # Main JavaScript file
│   └── modules
│       ├── nav.js      # Navigation functionality
│       ├── modal.js     # Modal functionality
│       └── carousel.js  # Carousel functionality
└── assets
    ├── fonts           # Directory for font files
    ├── svgs            # Directory for SVG files
    └── data
        └── content.json # Structured data for the website
```

## Getting Started

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the site.
3. Modify the CSS and JavaScript files as needed to customize the site.
